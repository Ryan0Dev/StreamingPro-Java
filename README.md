#  StreamingPro

Projeto desenvolvido com 50 dias de estudo de Java. O sistema simula uma plataforma de streaming completa.

# Funcionalidades
- **Cadastro Seguro:** Validação de nome completo e senha forte.
- **Gestão Financeira:** Entrada de saldo e cobrança automática de planos.
- **Segurança:** Trava de acesso (boolean flags) que impede o uso sem assinatura ativa.
- **Perfis Dinâmicos:** Uso de `ArrayList` para criar e gerenciar múltiplos usuários.

# Conceitos Aplicados
- Programação Orientada a Objetos (Comunicação entre classes).
- Estruturas de Repetição (`while`, `for`).
- Condicionais Complexas (`if/else` com `return`).
- Manipulação de Listas Dinâmicas.
